Шрифты для AMIGA IDE
====================

Рекомендуемый шрифт: JetBrains Mono
Скачать: https://www.jetbrains.com/mono/

Положи сюда:
- JetBrainsMono.ttf  (основной шрифт для редактора)
- console.ttf        (запасной вариант)
- ui.ttf             (шрифт для интерфейса, опционально)

Программа загрузит их автоматически!
